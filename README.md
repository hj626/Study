## 🎯 학습 목표: "Full-Stack Insight"
**데이터가 흐르는 전체 구조**를 설계할 수 있는 개발자가 되는 것이 **목표**입니다.

1. **Low-Level 이해**: C 언어를 통해 메모리 구조와 주소값의 원리를 파악하여 효율적인 백엔드 시스템 설계의 기초를 다지겠습니다.
2. **기술 융합**: C(Embedded/Client) + Spring Boot(Server) + RAG(AI)를 연결하는 **초소형 통합 프로젝트**를 구현을 목표로 학습하고 있습니다.
3. **지속적 확장**: Java와 유사한 C#을 습득하여, 다양한 환경에 유연하게 대응할 수 있는 풀스택 역량을 갖추고자 합니다.
