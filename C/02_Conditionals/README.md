<BR>

## 📝 오늘 배운 핵심 개념

**1. 조건문의 구조 (if, else if, else)**
* **if:** 조건이 참(true)일 때만 실행되는 가장 기본적인 제어문.
* **else if:** 여러 개의 조건을 순차적으로 검사할 때 사용, 앞선 조건이 거짓일 때 다음 조건을 확인.
* **else:** 제시된 모든 조건이 거짓일 때 마지막으로 실행되는 '기본값' 역할.

**2. 비교 연산자와 논리 연산자**
* `>` , `<` , `>=` , `<=` : 값의 크기를 비교하여 참/거짓을 판별.
* `%` (나머지 연산자): 짝수/홀수 판별 시 `num % 2 == 0`과 같이 활용하여 나누어떨어지는지 확인.
* `&&` (AND 연산자): 두 가지 이상의 조건이 모두 만족해야 할 때 사용. (예: `score >= 80 && score < 90`)
