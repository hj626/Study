# 📂 05. Arrays
>**학습 목표:** 동일한 타입의 데이터를 연속된 메모리 공간에 저장하는 배열의 개념을 이해하고, 
>반복문(for, while)을 활용해 데이터를 효율적으로 처리하는 방법을 숙달.
<BR>

### 📝 오늘 배운 핵심 개념

**1. 배열의 정의와 선언**
* **구조:** `자료형 배열명[크기];` 형태로 선언 (예: `int arr[10];`).
* **특징:** 배열의 인덱스는 **0부터 시작**하며, 마지막 인덱스는 `크기 - 1`임.

**2. 배열과 반복문의 조합**
* **활용:** `for`문이나 `while`문을 사용하여 배열의 모든 요소에 접근, 입력 및 출력을 자동화.
* **주의:** 루프 제어 변수(`i`)의 범위를 잘못 설정하면 배열 범위를 벗어나는 메모리 오류 발생 위험.

**3. 데이터 처리 로직 (최대/최소값/평균)**
* **평균:** 누적 합계(`sum`)를 구한 뒤 데이터의 개수로 나눔. 정확한 소수점을 위해 `10.0`과 같이 실수형 연산 권장.
* **최대/최소값:** 기준 변수(`max`, `min`)를 배열의 첫 번째 값(`arr[0]`)으로 초기화한 뒤, 비교 연산자를 통해 갱신.
