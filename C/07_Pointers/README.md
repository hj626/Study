# 📂 07. Pointer
>**학습 목표:** 변수의 주소를 저장하는 포인터의 개념 이해, <br>
>포인터를 활용해 변수의 값을 간접적으로 참조하고 변경하는 방법을 숙달.
---
### 📝 오늘 배운 핵심 개념
**1. 포인터의 정의와 선언**
* **구조:** `int *p = &a;` 형태로 선언.
* **특징:** `&a`는 변수 a의 메모리 주소, `p`는 그 주소를 담는 변수, `*p`는 p가 가리키는 주소로 가서 값을 가져오라는 의미.

**2. 포인터로 값 변경하기**
* **활용:** `*p = 20;` 처럼 포인터를 통해 원본 변수의 값을 간접적으로 변경 가능.
* **주의:** `int *p = &a;` 에서 `&a` 자리에는 반드시 주소가 들어가야 함. <br> 숫자 리터럴(예: `10`)을 넣으면 10번지 주소로 접근하게 되므로 절대 불가.

**3. 배열과 포인터의 관계**
* **핵심:** `arr[i]`와 `*(arr+i)`는 동일한 표현. 배열명 자체가 첫 번째 요소의 주소를 의미.
* **활용:** 포인터 변수를 배열에 연결하여 `*(p+i)` 형태로 배열 요소에 접근 가능.

**4. 함수의 포인터 매개변수**
* **구조:** `void swap(int *a, int *b)` 처럼 함수 선언 시 `int *a`는 int형 주소를 받을 준비가 된 매개변수 선언.
* **주의:** 이 경우 함수 호출 시 값이 아닌 주소(`&변수`)를 인자로 전달해야 함.
