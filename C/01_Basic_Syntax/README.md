# 📂 01. Basic Syntax & I/O
> **학습 목표:** C 언어의 기본 구조를 익히고, 데이터를 화면에 출력하거나 사용자로부터 입력받는 방법을 습득.
<BR>

## 📝 오늘 배운 핵심 개념

### 1. 표준 입출력 (Standard I/O)
* `printf()`: 데이터를 화면에 출력하는 함수.
* `scanf()`: 사용자로부터 데이터를 입력받아 변수에 저장하는 함수.

### 2. 변수와 자료형 (Variable & Data Type)
* `int`: 정수형 데이터를 저장.
* `char[]`: 문자열을 저장하기 위한 배열.
    * 예: `char name[20];`은 최대 19글자까지 저장 가능.

### 3. 주소값의 이해 (Address & Memory)
* 일반 변수(`int` 등)는 `scanf`로 값을 받을 때 `&` 연산자를 붙여 주소를 알려주어야 합니다.
* **중요!** 문자열 배열(`char[]`)의 이름은 그 자체로 주소값이기 때문에 `&`를 붙이지 않아도 됩니다.
