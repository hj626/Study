# 📂 04. Functions
>**학습 목표:** 반복되는 코드를 함수로 분리하여 재사용성과 가독성을 높이는 구조를 이해,
>반환값과 매개변수의 개념을 익혀 다양한 함수 실습을 통해 함수 중심의 사고방식을 숙달.
<BR>

### 📝 오늘 배운 핵심 개념

**1. 함수 정의와 호출**
* **구조:** `반환형 함수명(매개변수)` 형태로 정의하고, main에서 호출하여 결과를 사용.
* **활용:** 두 수의 합, 최댓값 반환 등 반복 사용되는 로직을 함수로 분리.

**2. 매개변수와 반환값**
* **매개변수:** 함수에 전달되는 변수로, 자료형 입력 필수 (예: `int a, int b`).
* **반환값:** `return`으로 결과를 돌려주며, 반환형이 `void`이면 반환값 없음.

**3. C언어의 참/거짓 (0과 1)**
* C언어에서는 `0`은 거짓(False), **0 이외의 숫자는 모두 참(True).**
* Java의 boolean(T/F)과 같은 개념으로, 함수의 반환값을 `if`문 조건으로 활용 가능.

--- 

**4. 변수의 종류**
| 구분 | 주요 내용 |
| :--- | :--- |
| **지역 변수** | main 안에 있는 변수 |
| **매개 변수** | 함수에 들어가는 변수 |
| **전역 변수** | 함수 밖에서 지정하여 어디서든 사용 가능한 변수 |
