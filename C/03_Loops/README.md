# 📂 03. Loops
>**학습 목표:** 반복적인 작업을 효율적으로 처리하기 위한 `for`문과 `while`문의 구조를 이해하고, 다양한 실습을 통해 조건에 따른 반복 로직을 숙달.
<BR>

### 📝 오늘 배운 핵심 개념

**1. for문: 정해진 횟수만큼 반복**
* **구조:** `for(시작값; 데드라인; 변화값)` 형태로 작성하여 반복 횟수를 명확하게 제어.
* **활용:** 구구단 출력이나 1부터 100까지의 합계 계산처럼 반복 범위가 정해져 있을 때 유용.

**2. while문: 조건이 참인 동안 반복**
* **구조:** 조건식이 참(`true`)인 동안 계속 실행, 루프 내부에서 조건을 변화시키는 코드(예: `num--`)가 반드시 필요.
* **활용:** 카운트다운이나 특정 목표치에 도달할 때까지 반복해야 하는 상황에 적합.

**3. 반복문과 조건문의 결합**
* 반복문 내부에서 `if`문을 사용하여 특정 조건(예: `i % 2 == 0`)을 만족하는 데이터만 골라 출력.
